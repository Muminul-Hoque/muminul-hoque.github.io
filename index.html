About Me

4chan Toxicity Analysis: Comparative Study Using OpenAI & Perspective APIs (2025)

[View on GitHub]

  • Analyzed toxic content on 4chan’s /pol/ board using OpenAI Moderation API and Google Perspective API.
  • Preprocessed posts, applied rate limiting, and stored structured JSON datasets.
  • Computed toxicity metrics (hate, harassment, sexual, threats, profanity) and compared model agreement.
  • Generated visualizations to highlight moderation differences between the two APIs.

Figure: Scatterplot comparing toxicity scores from OpenAI and Perspective APIs.
